## What's Inside? üìö
- **Core Math**: Set Theory, Logic, Algebra, Arithmetic, Relations, Number Sets
- **Advanced**: Calculus, Differential Equations, Linear Algebra, Geometry, Trigonometry
- **Sciences**: Physics Essentials, Chemistry
- **Greek Alphabet**: Full uppercase/lowercase with variants
- **Features**:
  - Instant search across all symbols
  - Copy LaTeX code with one click
  - Dark mode toggle
  - Export to PDF
  - Responsive on mobile, tablet, desktop

Every symbol includes: **Unicode**, **name**, **meaning**, **LaTeX/KaTeX code**, and a direct link to [Wolfram MathWorld](https://mathworld.wolfram.com).
